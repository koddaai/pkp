# @pkprotocol/spec

## 0.2.0

### Minor Changes

- Initial public release of PKP - Product Knowledge Protocol
  - @pkprotocol/spec: Zod schemas, parser, validator for 15 product categories
  - @pkprotocol/shared: Shared utilities (slugify)
  - @pkprotocol/static-generator: Generates .well-known/pkp/ structure
  - pkp (CLI): Commands for init, validate, build, serve, generate
  - @pkprotocol/catalog-server: MCP server for local product catalogs
  - @pkprotocol/registry-server: MCP server for global registry with PostgreSQL support
  - @pkprotocol/skills: Specialized AI prompts for product assistance
